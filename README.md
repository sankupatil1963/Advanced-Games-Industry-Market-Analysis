# 🎮 Advanced Market Analysis of the Games Industry using Python

## 📌 Project Overview

This project focuses on analyzing the gaming industry dataset using Python. The objective is to clean the data, perform Exploratory Data Analysis (EDA), conduct statistical analysis, and build a Machine Learning model to predict game user ratings.

The project demonstrates an end-to-end Data Analytics workflow including data preprocessing, visualization, hypothesis testing, and predictive modeling.

---

## 🎯 Project Objectives

- Perform Data Cleaning and Preprocessing
- Analyze gaming trends using Exploratory Data Analysis (EDA)
- Identify relationships between variables using Correlation Analysis
- Conduct Hypothesis Testing (ANOVA)
- Build a Machine Learning model to predict User Ratings
- Identify the most important features influencing game ratings
- Generate business insights from the analysis

---

## 📂 Dataset Information

The dataset contains information about video games, including:

- Game Name
- Genre
- Platform
- Release Year
- User Rating

- Sports games have the highest number of titles.
- PC contains the largest game collection.
- User ratings are concentrated between medium and high values.
- Game releases increased significantly over recent decades.
- Genre and Release Year are major factors influencing user ratings.
